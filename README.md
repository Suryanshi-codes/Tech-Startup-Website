# 🚀 TechNova - Responsive Tech Startup Website

TechNova is a modern and fully responsive startup landing page built using **HTML5** and **CSS3**. The project demonstrates responsive web design principles with a clean and professional user interface that adapts seamlessly to desktop, tablet, and mobile devices.

---

## 📌 Features

- Responsive Navigation Bar
- Hero Section
- About Section
- Services Section
- Pricing Plans
- Team Members
- Contact Form
- Footer
- Mobile-Friendly Layout
- CSS Flexbox
- CSS Grid
- CSS Media Queries
- Hover Effects
- Modern UI Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Google Fonts
- Font Awesome Icons
